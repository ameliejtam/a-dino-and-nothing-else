# Dino Game

A simple dinosaur game where the player controls a dino that jumps over obstacles and shoots fireballs at monsters.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd dino-game
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Running the Game

To start the game, run the following command:
```
python main.py
```

## Controls

- Press the **Space Bar** to make the dinosaur jump over obstacles.
- Double click the **Space Bar** to shoot a fireball at incoming monsters.

## Assets

- The `assets/dino` directory contains sprites for the dinosaur character.
- The `assets/obstacles` directory contains sprites for the obstacles.
- The `assets/fireballs` directory contains sprites for the fireballs.
